 stega
- это стеганограф с адаптивным алгоритмом скрытия информации. Он работает с форматами BMP и PNG и с сообщениями на латинице (вывод на кирилице - не читаем). В данной  програме реализованы две функции: hide - сокрытие информации и seek - дешифрование информации.

Синтаксис:

python stega.py hide <image> <текст> <выходной_файл> <размер_блока> [seed]

python stega.py seek <изображение> <ключ.txt> <размер_блока>
